# Assets

This directory should contain the following image assets:

## Required Files

- **icon.png** - 1024x1024px app icon
- **splash.png** - 1284x2778px splash screen (or any high-res image)
- **adaptive-icon.png** - 1024x1024px adaptive icon for Android

## Temporary Placeholders

For initial development, you can use simple placeholder images:

1. Create a solid color 1024x1024px image and save as `icon.png`
2. Create a solid color 1284x2778px image and save as `splash.png`
3. Copy `icon.png` to `adaptive-icon.png`

Or use online tools:
- https://www.figma.com (design custom icons)
- https://icon.kitchen (generate adaptive icons)
- https://appicon.co (generate all required sizes)

## Color Scheme

Match the app's dark theme:
- Background: #121212
- Primary: #4fc3f7 (blue)
- Text: #e0e0e0
